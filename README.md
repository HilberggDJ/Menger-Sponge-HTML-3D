# Gąbka Mengera

Interaktywna wizualizacja 3D fraktala w przeglądarce. Bez instalacji – wystarczy otworzyć `menger.html`.

---

## Obsługa

| Akcja | Efekt |
|---|---|
| Przeciągnij myszką | Obróć gąbkę |
| Suwak **Poziom** | Głębokość rekurencji (0 = kostka, 3 = pełna gąbka) |
| Suwak **Zoom** | Przybliż / oddal |

---

## Co to jest gąbka Mengera?

Fraktal 3D wymyślony przez Karla Mengera w 1926 roku. Budujesz go tak: weź kostkę, podziel na 27 mniejszych, usuń środkowe (7 sztuk), a dla każdej z pozostałych 20 powtórz to samo. Im więcej iteracji, tym więcej dziur.

---

## Technologie

Czysty HTML + JavaScript. Zero bibliotek.
